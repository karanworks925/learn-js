# Teaching Notes

## User Profile
- Goal: Land a developer job / become senior-dev ready
- Prior JS experience: Some — knows basics but has gaps; wants solid, complete foundation
- Focus: Interview-readiness + production code quality + senior-level understanding

## Preferences
- Full curriculum from base to every concept
- Wants to reach senior dev level

## Full Curriculum — 30 Lessons

### FOUNDATION (Lessons 01–06)
01. Variables, Types & Hoisting ✅
02. Operators & Control Flow
03. Functions — Declarations, Expressions, Parameters
04. Loops — for, while, for...of, for...in, iterating patterns
05. Arrays — Basics and Core Methods
06. Objects — Basics, Methods, Iteration

### INTERMEDIATE (Lessons 07–12)
07. Strings & Template Literals
08. Scope & Lexical Environment
09. Closures — The Most Important JS Concept
10. The `this` Keyword — 4 Binding Rules
11. Higher-Order Functions — map, filter, reduce
12. Destructuring, Spread & Rest

### INTERMEDIATE-ADVANCED (Lessons 13–18)
13. Prototypes & the Prototype Chain
14. ES6 Classes & Inheritance
15. Error Handling — try/catch/finally, Custom Errors
16. Modules — import/export, CommonJS vs ESM

### ASYNC & DOM (Lessons 17–22)
17. The Event Loop — Call Stack, Task Queue, Microtasks
18. Callbacks & Promises
19. async/await — Modern Async JS
20. DOM Manipulation
21. Events & Event Delegation
22. Fetch API & Browser Storage

### SENIOR LEVEL (Lessons 23–27)
23. Functional Programming — Pure Functions, Composition, Currying
24. Design Patterns in JavaScript
25. Iterators, Generators & Symbols
26. Proxy, Reflect & Meta-programming
27. Memory Management & Performance

### EXPERT / INTERVIEW READY (Lessons 28–30)
28. Security in JavaScript
29. Testing JavaScript — Unit Tests, TDD, Jest
30. Interview Patterns & Algorithm Fundamentals
